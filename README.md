# RainbowDrawingTools

RainbowDrawingTools is a set of tools for creating dynamic, colorful drawings in graphical applications. With the RainbowPen and RainbowBrush classes, users can draw with pens and brushes that automatically change colors while drawing, creating a rainbow effect on the screen.

Features:

- Generate rainbow colors and transitions between any two colors.

- Create pens and brushes that dynamically change color as you draw.

- Support for different orientations and fill modes, such as stretch, repeat, and bounce.

- Ability to create advanced, colorful effects in drawing applications, such as lines, rectangles, and ellipses.

- Ideal for those looking for creative solutions to produce dynamic, colorful drawings, both in educational and artistic applications.
